# GOWS Homepage: Claude Code 運用ガイド

このリポジトリでは、Claude Code を使ってGOWS合同会社のホームページを構築する。

## GOWS合同会社の概要
### 主な事業
- ITコンサルティング（特にDX・AX）
- システム開発
### サブ事業
- 自社サービス運営

## 基本方針
- 不明点がある場合は、推測で実装せず確認事項を提示する

## ルール
- ファイルの追加等は自由に行なって良い

## 作業プロセス
1. corporate-engagement-specialist を使って構成や文章等の内容を作成
2. 1の結果を元に web-designer を使ってデザインを作成
3. デザインを元に homepage-builder を使って実装
